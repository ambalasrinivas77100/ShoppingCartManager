# Shopping Cart Management System

## Description
A Java console-based shopping cart application developed using OOP concepts and Java Collections.

## Features
- View Products
- Add Products
- Merge Duplicate Items
- Update Quantity
- Remove Products
- Cart Summary
- Checkout

## Technologies
- Java
- Eclipse IDE
- ArrayList
- Iterator
- OOP

## Project Structure

src/
 └── shoppingcart/
      ├── Product.java
      ├── CartItem.java
      ├── ShoppingCart.java
      └── Main.java
